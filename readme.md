# usage

1. 项目根目录创建 .env文件

   ```env
   qq=3481477273
   token=eyJhbGciOiJxxx
   ```

2. 执行如下命令（提示安装tsx确认即可）

   ```bash
   pnpm install && pnpm dev
   ```
   也可以使用其它的包管理工具

3. 扫码登录即可
